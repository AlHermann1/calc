# Calc

A simple calculator app built with Qt and C++.

---

## Running the Prebuilt Application (Windows only)

1. Go to the [Releases](https://github.com/AlHermann1/calc/releases) page.
2. Download the **`bin.zip`** file.
3. Extract the contents of the zip file.
4. Open the folder `install/bin/` and run `calc.exe`.

## Mac/Linux 

1. Download Qt version 6+
2. Download the source code
3. Build project with cmake
